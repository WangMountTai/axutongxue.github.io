上网能解决问题，但如果问题就是无法上网，怎么办？

发表于 2018-12-28 | 分类于 公众号 | | 阅读数

如今有了网络，面对很多问题

我们可能都是：先上网搜解决办法~

所以啊所以啊，为了避免上述情况，就得做好预防措施！

一、俗话说有备无患

阿虚常备了不少网络修复软件，就是为了避免上述情况出现。

建议大家也是常备今天提供的软件！

下面这些软件可以用于修复：

  • 连不上WI-FI啊
  • 登录不上QQ啊
  • 能上QQ但是打不开网页啊
  • 某些网页无法正常打开啊
  • 或者其他问题

首先说明

当然，软件能解决的问题仅限于系统设置问题。

这些问题肯定是靠软件没办法解决的：网线松了，路由器坏了，宽带欠费了….

1.360断网急救箱

首先推荐使用这个软件，虽然360很辣鸡，但提供的某些小工具还是挺好用的。

本工具是从360中提取的单文件版，可以修复普通人遇到的大部分问题。

使用方法：下载后直接运行360断网急救箱单文件版.exe即可。

软件下载方式见文末

2.网络连接修复工具

使用方法：无需安装，解压后直接运行lspfix_10.exe

软件下载方式见文末

以下是作者对软件的介绍：

近期有不少朋友遇到了一些网络问题，比如网络通畅，但是本机的网络适配器就是一个红X，无线设备也不无法搜到无线信号，无法正常上网。

遇到这种情况重启电脑也无效，怎么设置网络协议都无用。

这类问题通常出现在使用加速器、Ⅴρn或某些特殊的网络软件之后。

其实解决方法很简单，只要重置winsock协议就能解决大部分问题。就是以管理员权限运行命令行，执行2个命令的事。

起初我遇到的时候，手动处理一下就好，但是后来又遇到了几次，也就懒了，随手拿起原来的一段代码改改内容换个皮，做成一键式的修复工具。

使用简单，一键搞定，各位遇到此类问题的朋友可以留着尝试。

原文地址：http://www.hxlive.cn/archives/655

3.百度、360、金山LSP修复工具

LSP的中文名叫分层服务提供程序，是TCP/IP等协议的接口。

  • 浏览器，聊天工具等等都要通过这个接口来获取相应的信息，LSP的功能就相当于生活中的插座。
  • 当LSP出现异常时，这个调用的接口就没办法正常工作，就会出现QQ能上网页打不开的情况。

还有种解释是：Windows 的网络连接可以由称为“分层服务提供商(LSP)”的机制进行扩展，但这种扩展经常会被一些病毒或恶意程序利用，劫持LSP协议，导致的结果是所有与网络交换的信息都要通过这些恶意软件，从而使得它们可以监控使用者的信息。

百度

使用方法：运行百度LSP修复文件夹下的LSPFixTool.exe

软件下载方式见文末

360

使用方法：运行360LSP修复文件夹下，Utils文件夹中的LSPFix.exe

软件下载方式见文末

金山

使用方法：运行金山LSP修复文件夹下的klsprepairer.exe

软件下载方式见文末

4.Complete Internet Repair

当你用完以上工具还无法解决，请尝试Complete Internet Repair ，这款工具基本能够搞定绝大多数的网络问题。

使用方法：无需安装，下载后直接运行Complete Internet Repair单文件汉化版 ，勾选掉需要修复的选项，然后点击开始，直接进行修复。

软件下载方式见文末

注意

同时也要提醒大家的是：因为每个人遇到的问题不尽相同，使用此工具前，请先百度一下你遇到的问题可能是什么。

再用此工具针对性的进行修复，没有问题的功能不要尝试修复，以免修复出问题了。

软件主页：https://www.rizonesoft.com/
