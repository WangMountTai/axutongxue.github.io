国内我最想去的地方，是台湾

发表于 2018-12-02 | 分类于 公众号 | | 阅读数

我还没去过台湾，但我一定会去。

一、关于台湾的印象

我心中的台湾，是慢慢积累的。

是从小到大认识的偶像男子组合、作家、诗歌、电影、歌手…

1.偶像男子组合

最早的台湾印象，来自棒棒堂，一个台湾偶像男子组合。

十多年前，还在读小学的我。认识了他们。

  • 2006年12月2日，棒棒堂确定成军。

  • 2007年，发表专辑《七彩棒棒堂》并踏上台北小巨蛋，完成首场演唱会，且出演了电视剧《黑糖玛奇朵》。

  • 2008年，发表专辑《哪里怕》和《藏经阁》，并在小巨蛋举办梦想出发闪耀演唱会，同年出演《黑糖群侠传》电视剧，获得蒙牛酸酸乳音乐风云榜新人盛典 - 年度港台最佳新人组合。
  • …….

我和我那时那群哥们，一起追《黑糖玛奇朵》、《黑糖群侠传》，一起唱《哪里怕》、《藏经阁》…

回头到现在，全是美好的回忆。

2.作家

应该是2010年，在图书馆无意读了九把刀的《月老》。

第一次认识了九把刀，台湾网络作家。

他写过《等一个人咖啡》、《爱情，两好三坏》、《后青春期的诗》、《月台》、《那些年,我们一起追的女孩》、《杀手系列》…

  • 《等一个人咖啡》中的咖啡店「等一个人」
  • 《爱情，两好三坏》中的棒球打击场
  • 《后青春期的诗》中毕业那年学校后埋下的约定
  • 《月台》中时空暂停的“零时”月台
  • 《那些年,我们一起追的女孩》中彰化精诚中学那面涂满涂鸦的墙壁
  • 《杀手》中泰利台风过境台湾的那个晚上

从他的书中，处处是台湾的影子。

3.诗歌

赋予台湾文艺气息的，是我记忆中关于台湾的诗歌：

是闻一多先生的《七子之歌·台湾》

我们是东海捧出的珍珠一串，
琉球是我的群弟，我就是台湾。
我胸中还氤氲着郑氏的英魂，
精忠的赤血点染了我的家传。
母亲，酷炎的夏日要晒死我了，
赐我个号令，我还能背城一战。
母亲，我要回来，母亲！

是余光中先生的《乡愁》

小时候，
乡愁是一枚小小的邮票，
我在这头，
母亲在那头。

长大后，
乡愁是一张窄窄的船票，
我在这头，
新娘在那头。

后来啊，
乡愁是一方矮矮的坟墓，
我在外头，
母亲在里头。

而现在，
乡愁是一湾浅浅的海峡，
我在这头，
大陆在那头。

是台湾诗人郑愁予先生的《错误》

我达达的马蹄是美丽的错误，

我不是归人，是个过客……

是台湾诗人席慕蓉女士的《乡愁》

故乡的歌是一支清远的笛
总在有月亮的晚上响起

4.电影

台湾的美景，在我记忆中的一部部电影之中徐徐展开。

电影毫无疑问成了最好的台湾宣传片

事隔经年，这些电影仍留在自己的记忆中。

5.歌手

不得不说的是：我最爱的两个歌手：吴青峰、林宥嘉。

都是台湾人。

喜欢吴青峰，从一首「小情歌」开始。那是2006年。

喜欢林宥嘉，从一首「残酷月光」开始。那是2008年。

▲曾为他们做的海报

二、非去不可的梦想

不知不觉中，关于台湾的点滴印象慢慢汇聚，竟产生了「非去一次不可」的汩汩念头。

人为什么活着？

高中时候，我看了一部短片，名为《不老骑士》

故事讲述了：

五个台湾老人，平均年龄81岁，一个重听，一个得了癌症，三个有心脏病，每一位都有退化性关节炎。

然而，他们梦想环岛一圈。

6个月的准备，环岛13天，1139公里，从北到南，从黑夜到白天…

只为了一个简单而伟大的理由…

为了梦想。

完整纪录片88分钟，爱奇艺可看。

人为什么活着？——短片的答案是：梦。

那去台湾旅游一次，就是我的一个梦。

看完短片，更对台湾人有了一种莫名的情愫。

都说台湾最美的风景不是阿里山、日月潭或者太鲁阁，而是台湾人。

  • 我想找当地人学学台湾腔；

  • 我想在整洁的街道上压马路，听密密麻麻的机车队轰鸣的声音；

  • 在捷运里偷听湾湾的青年人用嗲嗲的台湾话聊着有趣的话题；

  • 看卖臭豆腐的帅气台湾男生认真切豆腐的模样；

  • 去南投溪头妖怪村邂逅一群“妖怪”；

  • 坐一坐海天一色的南回线铁路，然后在垦丁为冲浪阳光老男孩加油助威。

所谓旅行，实际上就是去了解当地的文化，我们大多数人的旅行恐怕竟是了解吃的文化、喝的文化。

而我，想去台湾，感受人的文化。

如果可以，我也想结交一个台湾的朋友。

三、想介绍我的一对朋友

为什么要说一对？

因为他们是一对双胞胎兄弟。

姑且在这里称呼他们为林琛兄弟吧。

现实中我们是老友，一直到大学前，我们都是同学。

我很羡慕他们，不为别的，就因为他们做了我想做却没做到的事——20岁以前，一场说走就走的旅行。

第一站：广州

下一站是深圳

然后是香港

再来是厦门

辗转上海

最后是武汉

他们说：6座城市，24W步，出发并没有想象当中那么难

我自己是暗自钦佩的。

后来的他们，甚至又来了一场说走就走的旅行，这一次，是去印度尼西亚。

我想

如果你想详细了解他们的过往经历

如果你也爱摄影或者说爱旅行

又或者说你准备有一场旅行

相信你也会喜欢他们

他们在这里，分享他们对世界的看法，告诉别人——他们是怎么样凝望这个世界的。

四、再回到我去台湾的愿望

虽然说旅行这东西的诀窍在于：心血来潮马上动身。

但是，我还想等一等。

等完全靠自己赚够旅行的启动金，等一段足够充裕的时间，等一个志同道合的朋友，又或者说，等一段感情。

总有一天

  • 总有一天，我要去到台湾，去九把刀书中无数次提到的彰化打卡。

  • 总有一天，我要去到台湾，到《海角七号》的取景地垦丁，去踩一下白砂湾。

  • 总有一天，我要去到台湾，到《听说》中士林夜市，去好好吃一吃美食。

  • 总有一天，我要去到台湾，听一场苏打绿、听一场林宥嘉。

  • 总有一天，我要去到台湾，友人相伴最好，一个人独行也罢。

  • 总有一天，我要去到台湾……

或许？会约上他俩兄弟一起。

生活很忙，未来很长。我想在我还年轻的时候，完成这个梦想。

最好，能在三十岁前。

我梦想做到：凡心所向，素履以往。——努力的去做。

因为有梦想，人生才会有方向。

生于浊浊之世，就算仍未动身，我也想先好好凝望一下这个世界。
