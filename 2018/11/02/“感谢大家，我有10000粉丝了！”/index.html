“感谢大家，我有10000粉丝了！”

发表于 2018-11-02 | 分类于 公众号 | | 阅读数

一首来自Motohiko Hirami的冷门轻音《Prelude》。

空灵的旋律，就像蝉鸣声中树荫底穿隙而过的夏 。曲中孩子们的嬉戏打闹让我想起小时候的天真、快乐、自由，儿时我们都是这么欢声笑语，如今长大了，更多的却是愁眉不展。可以的话，放慢脚步，静下心听听。

所以今天，又想静下心来写写心情。

无意间听到这首曲子，第一次，就给了我一种命中相遇的感觉。

35秒，它呼唤着我。

一、小目标

昨天晚上，再次查看了一下粉丝数。恍然发现，公众号粉丝已经破1W了！

惊讶之余却也感到理所当然啊，因为公众号每天都在稳步涨粉，自己也每天都在查看。

这10000+粉丝，是你、是他、是她……是由关注着「阿虚同学」的各位组成的。

原本阿虚心里有个小目标：就是在2018年末，能达到10000粉丝。

现在来看：小目标还是要有的，万一就实现了呢。๑乛◡乛๑

  • 其中，得感谢不少粉丝的帮忙推广！也得感谢各位的帮忙转发！

  • 某次，有粉丝在「豆瓣鹅组」上帮忙推广了一下公众号，一下就引来了上百人关注，一下子让阿虚也很吃惊呢。毕竟很少有一天能突然增加几百粉丝。

二、回首

  • 2018年2月18日，我注册了“阿虚同学”。
  • 2018年7月31日，公众号粉丝破5000，开通了流量主。
  • 2018年11月1日，公众号粉丝破10000。

公众号粉丝破5000后不久，我写了一篇推送：《音乐故事丨“想和你聊一聊，你愿意听吗？”》https://mp.weixin.qq.com/s/lQQ_pVyN3LRGrlE8NXoK2g

为什么会写那篇推送？

推送内容大致是我做公众号的目的和打算，那是当时我对自己的一些审视和思考。

另外一些原因是这样的：

  1. 「阿虚」本是我很喜欢的一个二次元人物的名字，出自《凉宫春日的忧郁》，是部多年前的作品了，整部小说&动漫都没有提到阿虚的真名。

  2. 公众号命名为「阿虚同学」，一是为了注册用户名不与别人重复，二是我想给公众号一个形象。

  3. 不时的分享我运营公众号的心路历程，是想要让「阿虚同学」这个公众号显得有血有肉。

  4. 又要提到「可能吧」的一句话了：「最糟糕的营销是，你记住了故事，却忘记了品牌」，我想让大家记住的是「阿虚同学」这个形象、这个品牌。某一天，这个公众号可能不在了，但是你可能依然能在互联网上找到「阿虚同学」的足迹。

三、再次审视

细心的小伙伴可能会发现，我又重新给了公众号一个简介。

一个特立独行的95后青年，爱音乐，爱折腾，爱分享。

我曾用过另外两个词来形容自己：我行我素、独来独往。最后还是觉得特立独行比较合适，但自己多少有点我行我素、独来独往吧，不然也不会想到这两个词。

音乐

是我的另一半灵魂。有人说：孤独的人有酒，有酒的人孤独。我大抵身体原因是会戒酒的，所以换做是我，我想说，孤独的人有歌，听歌的人孤独。

折腾

这个词很适合我，但其实我想说，我爱折腾，爱折腾，爱折腾！在某些时候，在让自己满意之前，我一定会去反复折腾。这，也是我能知道这么多杂七杂八的知识的原因之一吧。

分享

无人享而不欢。知识只有分享了才有意义，阿虚分享的很多东西，也不过是别人分享出来的，阿虚只是做一个传播者而已。

懂一点技术

我知道关注我的人很杂，因为我分享的东西很多：科学上网、百度云破解、VIP视频破解、百度文库破解、音乐、省钱、摄影、阅读、漫画等等……

这些杂七杂八的推送，都是我曾经解决过的问题。我很难定义我擅长什么，或许就只是什么都懂一点吧。

但这些分享的东西都是对我的生活起到了一定帮助的，我想，若能对你有帮助，那就更好了。

我已经写了不少教程和分享了，你可能是从各个地方关注到的我。

我希望新关注我的粉丝，可以翻一翻公众号的目录，往期的推送，都能找到。

  • 我想在互联网上留下点什么，留下点真正有用的东西，是那些即是过了几个月、几年，拿出来依然受用的东西。
  • 我想你不论什么时候关注到我，总能学到点、收获点东西。
  • 我想那些无意间关注到我的粉丝，有种发现宝的感觉。

知一些道理

前5000粉丝，花了将近6个月的时间，后5000粉丝却不到3个月。

我认识到了这叫「积累」，这叫「冰冻三尺非一日之寒」。

很小的时候，我认定一句话：「坚持就是胜利」。

很俗，但是很实在。

有时候，我们不需要多高深的哲理，我们只需要相信一些最简单的东西。认定它是对的，并付之行动。

人生不需要多么高深的哲理，生活亦处处是道理。

文章比较小众，观点不很客观，谨慎关注。

文章比较小众：

前面也写到了，阿虚只是做了一个小小的个人公众号，关注阿虚的人又很杂。

  • 所以必然每次的推送无法照顾到每个粉丝的需求，但总的来说，阿虚的推送都比较干货，都会比较贴近生活、比较实用。

  • 因为阿虚比较爱折腾，所以就算是各种各样的小众问题，我都喜欢折腾出一个令自己满意的解决方案。

说不定某一天的推送里，正好也能帮你解决同样的问题呢？

观点不很客观：

我选择对这个世界保持自己的看法。

  • 写下的很多教程只是主观体验，可能帮不到你；
  • 会不时给出阿虚自己的一些观点和想法，可能与你有悖；
  • 会不时提到最近社会上的一些事情，可能不很客观。

如果，你觉得我分享的教程、观点、资源对你没有帮助，请谨慎关注。

所以，反过来想，其实写博客可能才更适合我。只是在当今的快媒体时代，能静下心来写博的人已经很少了。不少曾经的博主都转头开起了公众号。

毕竟，大部分博主写博还是希望能有人看见的。

说起来正奇怪呢，我是先做的公众号，之后才准备写博客的。这大概就是我所谓的特立独行吧。

四、后记

还有一些想说的

某一天，公众号可能会被封

可以给大家看一看公众号的部分违规记录。

个人做号真的不易，阿虚基本上没接广告，写了70多篇推送，一共就接了2次广告，并且都不是头条推送，然而就是这样，依然有人举报。

我想说，如果我发的东西你不喜欢看，取消关注即可，何必举报呢？

做一个善良的人吧。

之前写的一篇关于百度云破解的推送，我本有在知乎上同步发送，无奈知乎无论如何都会判为违规，所以最后也没能发出来。现在那篇文章依然在公众号历史消息，我并没有删除。但是某一天，可能也会像上面一样，被人举报，然后公众号被记录违规。

我也知道，我发的某些东西，是不太上台面的。但我也知道，一部分人是需要这些东西的。

我也不知道公众号能走多远，还是那句话：且行且珍惜吧。

一个人做公众号

最近学到一个新词：「李荣浩」

一个人做公众号呀，可能有点像李荣浩吧。

编辑、排版、制作封面、校对、多平台发布、重新排版等等，这些都是一个人在做。

还得回复后台消息、群消息，不时还有人来QQ咨询。

阿虚也需要时间去学习和休息，所以，某些时候回复的态度如果不好，请见谅呢。

很多问题我之前都写有教程，所以，如果我直接回复让你去翻一下往期教程，也请理解。

感谢

很高兴，一直有许多粉丝在默默的支持着阿虚，尽管没办法开通留言功能，但我能从每篇推送的点赞数看出来。

大家在公众号上发送的消息我也都是能看到的，下面只截了最近的部分后台消息：

阿虚也知道，有许多粉丝在默默帮忙点广告。

阿虚也看到，每篇推送都能收到赞赏，而这，对于阿虚来说，是一种感动。

最后

阿虚想说：感谢是相互的，阿虚也很感谢大家。

谢谢各位的关注，阿虚也只是做了小小的一点工作而已。
