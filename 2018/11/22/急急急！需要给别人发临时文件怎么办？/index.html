急急急！需要给别人发临时文件怎么办？

发表于 2018-11-22 | 分类于 公众号 | | 阅读数

你在某个平台上遇到一个陌生网友，

你没有没有他的QQ/邮箱/微信，

需要给他分享一个文件，

你怎么办？

突然！临时！需要分享个文件？

一般情况，你可能是选择下面两个办法

1.用百度云/微云等网盘

首先，这些网盘一般存在严重的限速。

不开会员的情况下：你上传限速不说，别人下载也限速。

另外，你可能只是想临时分享一下，并不想它占用你的网盘空间。

而且也懒得别人下载后，自己又去删除临时分享的文件。

2.用QQ传文件？

先说别人可能不使用QQ呢？

就算你们同时在线，你也加了别人QQ。

然而QQ传文件也是有限速的，一般情况下在600-1000k/s左右。

好像问题不好解决？

正文！正文！

因为阿虚经常给别人分享文件，所以今天来分享一下我觉得最优的解决办法。

首先首先！下文提到的方法均可以满足：

  1. 不限速上传
  2. 不限速下载
  3. 能获得临时的下载链接
  4. 文件过期后自动删除

一、各种邮箱

指139邮箱、163邮箱、新浪邮箱、QQ邮箱等。

为什么首推用邮箱来发送临时文件？

因为真的很好用！

  • 首先是安全性，相对其他云存储服务，邮箱的安全性绝对是排在前面的。

  • 而且现在浏览器普遍有自动填写帐号密码的功能，所以也不嫌登录麻烦。

阿虚常用139邮箱、163邮箱

1. 移动139邮箱

网址：https://mail.10086.cn/

中国移动提供的手机邮箱，只要你有移动手机号，就有此邮箱帐号。

登录邮箱，点击彩云网盘—暂存柜，即可上传临时文件。

  • 单个文件限制上传大小为2G
  • 文件保存期为15天

获取分享链接方法：

选中上传的文件，点击分享按钮，即可获取临时分享链接。

2. 网易163邮箱

网址：https://mail.163.com/

网易邮箱应该大部分人都有，手机号+@163.com，就能注册一个网易163邮箱。

  • 临时空间15G
  • 单个文件限制上传大小为2G
  • 文件保存期为15天

获取分享链接方法：

点击「下载」后，即会跳转到下载页面，复制浏览器的地址，即是临时下载链接地址

3. 新浪邮箱 和 QQ邮箱

新浪邮箱和QQ邮箱无法获取文件分享链接，只能把上传的文件作为邮件附件发送。

所以需要知道别人的邮箱地址才能发送文件。

新浪邮箱网址：https://mail.sina.com.cn/

  • · 单文件大小限制为2G
  • · 中转站文件7天有效，如需继续保存可以续期
  • ·文件在发送前可设置下载密码

QQ邮箱网址：https://mail.qq.com/

  • 临时空间2T
  • 单个文件限制上传大小为2G
  • 50M以下文件永久保存，50M以上文件保存期为30天

二、免登陆临时发送文件

以下推荐的是无需登录即可上传文件进行临时分享的几个网站。

1. 火狐

网址：https://send.firefox.com/

火狐浏览器推出的服务，文件的安全性还是很高的。

  • 无需注册登录即可使用
  • 应该为无限空间
  • 单个文件限制上传大小为1G
  • 每个链接将在 1 次下载或 24 小时后过期

2. TMP LINK

网址：http://tmp.link/

这是V站站友开发的一个网站，优点比较多。

  • 无需注册登录即可使用

  • 无限空间

  • 若分享的是视频与音频，分享页面可在线播放预览。

  • 单个文件限制上传大小为5G
  • 分享方式1，临时链：有效期24小时，24小时之后将失效。
  • 分享方式2，热力链：有效期72小时，当分享的文件在有效期内有人下载时，自动刷新有效期，有效期结束后失效。

阿虚有时分享音乐或者视频的时候会用到。

3. BITSEND

网址：https://bitsend.jp/?setLang=zh-tw

虽然是个日本的网站，但是提供了繁体中文页面。

  • 无需注册登录即可使用

  • 可以设置高级点的保存时间，分享次数和下载次数等

  • 单个文件限制上传大小为3G
  • 文件最多可以保存14天

4. 奶牛快传

网址：https://cowtransfer.com/

  • 无需注册登录即可使用

  • 单个文件限制上传大小为2G

  • 14天文件保存时间（未登陆用户为7天）
  • 无文件大小限制（未登陆用户为2G）
