教你下载被迅雷&百度云屏蔽的资源（阿虚独家心得）

发表于 2018-10-26 | 分类于 公众号 | | 阅读数

今天教一下如何下载各类被屏蔽资源

你是否遇到过以下情况

百度云：因含有违规内容，被系统屏蔽无法下载

迅雷：应版权方要求，文件无法下载

迅雷：敏感资源，文件无法下载

阿虚首先要说

针对以上情况，并没有万能的解决办法，阿虚只能提供一下思路，主要是学会方法。

首先要搞清楚

下载网上一个种子or磁力链接

主要分大概这几类下载方法

  1. 用迅雷下载
  2. 用其他BT下载器
    1. 百度云离线下载
    2. 国内其他离线下载
    3. 国外离线下载

离线下载，即利用服务器“替”用户下载，帮你下载好之后，你可以再去下载或者在线观看等。

所以，如果在上述下载方法中，任何一个出现下载不了的问题

解决办法

  1. 换用其他渠道
  2. 换下载的资源（换种子、换磁链）

请明白：

  • 如果下载的是死种&死链，即无人做种（这种一般是多年前的资源），那么就算更换下载渠道，一样的无法下载的。
  • 这种情况请换种子&磁链（尽量找近期发布的资源）

科普一下：什么叫做种？

  1. 发布BT种子的人，做种多少天指的就是持续多少天不撤种。这期间如果有别人下完了，就叫出种。下完的人可以继续做种。这时发布种子的人就可以不做种了，叫撤种。
  2. 一个资源只要有一个人在做种，那就其他人就可以继续下。如果没人做种了，叫断种，这是资源就死了，下不了了。某人长时间一直做种，叫保种。
  3. 做种的人越多，下载速度就越快，所以BT提倡大家都能积极保种，这样BT才能健康发展。
  4. BT上传的时候，数据会上传给其它所有正在下载同一资源的人。

1. 用迅雷下载

使用阿虚提供的老版迅雷：迅雷5，即可以解决大部分此类问题

如果你是迅雷会员，建议使用hosts屏蔽

这个方法优点是可以正常下载HTTP、HTTPS资源，还可享受大部分资源加速服务，但如果使用全局代理软件则有可能会失效！

  1. 用记事本打开hosts文件,即 C:\Windows\System32\drivers\etc\ 下 hosts 文件 (如不会请通过搜索解决)

  2. 添加以下三段到hosts文件内,并直接保存hosts

    127.0.0.1 hub5btmain.sandai.net

    127.0.0.1 hub5emu.sandai.net

    127.0.0.1 upgrade.xl9.xunlei.com

  3. 使用快捷键 win + r 打开运行窗口，然后输入 cmd, 回车

  4. 输入：ipconfig /flushdns, 回车, 即可刷新hosts文件, 使其立即生效

2. 用其他BT下载器

  • 所以你用其他软件下载遇到没速度是正常的。（尤其是冷门资源）

  • 如果你下载的是热门资源，但是被迅雷屏蔽了，建议试试下列软件

5. 国外离线下载

阿虚只能遗憾的说，国外离线下载并不理想。

一是从离线成功率来说，二是从离线成功后下载到本地的速度来说。

  1. 理想成功率取决于资源热度，很多时候无法成功离线下载（尤其冷门资源、几年前的种子）
  2. 由于是国外站点，离线成功后下载到本地的速度往往不会很快
  3. 加之免费用户限制多，你没有国外信用卡 / Pay pal等，想付费使用国外的服务，也很困难

介于以上三点，所以阿虚就不详细推荐了

5.1 国外有哪些离线下载站点

至于有哪些国外离线站点，有很多，比如Seedr、Furk 、OffCloud

复制粘贴的事阿虚不喜欢做，给大家指两条路：

  1. http://www.138vps.com/freeot/1299.html
  2. https://sofun.tw/bt-download-service/

以上两个博主已经把国内外的离线下载服务列举的差不多了

5.2 那国外离线下载有什么用？

上面又说下载成功率低，又说就算成功后下载会本地速度不理想，那么阿虚为什么要说这个方法？

阿虚教一个办法：

  • 比如你在Seedr离线下载完成后，从Seedr下载回本地很慢。但是你别忘了百度云啊！你可以再把Seedr的下载地址，放到百度云离线！

  • 尽管速度会很慢…一般可能只有几十KB，但反正也不用自己挂机下载是不是？所以这个下载过程其实没想象的那么「艰辛」

  • 今天把离线任务扔上去，睡一觉起来百度就帮下载好了，然后你再从百度下载到本地就行了（由于百度云在国内的优势，下载速度就要快很多了。）

不是每个站点离线成功后的下载地址都可以拿到百度云离线下载的，阿虚这里只是拿Seedr举例

等于说是百度云无法直接离线下载的文件，我们用国外离线下载中转一下，就可以用百度云离线下载了。
