不管手机还是电脑，换掉系统自带的视频播放器吧！

发表于 2019-01-05 | 分类于 公众号 | | 阅读数

提到视频播放器，请一定不要想到什么腾讯、优酷、爱奇艺视频，这些客户端无一列外的广告多+占用内存高+播放卡顿。

当然也不推荐window media player（系统自带的视频播放器），不能正常流畅播放视频的情况数不胜数。

无论哪个平台，你都需要一个强大的本地视频播放器。

如果你对视频播放背后的机理有一定的了解，应该会知道如果要播放某段视频，播放软件所需要做的：

  1. 首先播放器要支持某种容器（在视频文件上表现为MKV、MP4 等格式）
  2. 然后把视频流（H.264、H. 265 等）、音频流（FLAC、DTS等）等内容从容器中分离出来
  3. 将视频、音频「压缩的」编码数据，解码成为「非压缩的」视频、音频原始数据
  4. 然后渲染，就可以播放出视频、音频乃至字幕等内容了。

因此，分离器、解码器和渲染器等滤镜的质量，从根本上决定了一个视频播放器播放视频能力的高低、效果的好坏。

你可能有些时候会遇到：播放有画面没声音，或者说有声音没画面，大多数情况都是播放器的解码器不行。

前言

本篇文章单纯是软件推荐，核心目的是：让你知道有这些软件

再之，以下软件的评测和介绍网上已经数不胜数，阿虚不想做复制粘贴一类的事情，所以一概简单介绍。

况且，对于此类软件的介绍充斥着大量专业名词，诸如：

  • 多线程H264、MPEG-2解码器。(ffmpeg-mt)
  • DirectShow分离器和解码器支持。
  • 基于DirectShow和VSFliter的双字幕支持。
  • PMP分离器，优化部分解码器优先级。
  • TS、TP等高清视频的兼容性更好。

对于一般的用户来说，想必一点都不会去关心这些东西。

我们一般的用户在使用播放器方面，其实没有太多的要求，就是能够播放的格式多、打开速度快、资源占用小就行了，最后没有广告，那就是最好的了！

如果以下阿虚推荐的软件，你都没有用过，那就任选一款播放器。

如果某一款播放器你已经用着满意了，就没必要频繁更换了！

  • 前排提醒，下方教程充斥着大量网址。
  • 点击文末「阅读原文」可获得最佳阅读体验。

一、Windows

先说自己，这么久以来，阿虚电脑上只依次换用过三款视频播放器：QQplayer→KMPlayer→PotPlayer

PotPlayer

官网下载1：https://potplayer.daum.net/

官网下载2：http://www.potplayer.org/

无论你在网上任何地方问Windows上好用的播放器，PotPlayer一定榜上有名。

PotPlayer 的优势在于强大的内置解码器，它的前身是 KMPlayer，原作者姜勇希在跳槽到 Daum 公司后使用 C++ 完全重写。

PotPlayer 同时简化了内部的解码器体系，完成了 KMP 无法实现的 DXVA 硬件解码以及多线程解码功能，总之来说就是解码功能更强大了。

PotPlayer 并非是开源软件，但它也不存在丧心病狂的广告，并没有什么多余的后台进程，很是干净。

  • 对了，不得不说的是PotPlayer可以实现 0.2—12倍变速，并且变速不变声！
  • 阿虚常喜欢用PotPlayer1.2倍速追剧呢。

  • PotPlayer 最新中文版下载 - 最强万能格式影音视频播放器 (高清电影流畅/效果优异) https://www.iplaysoft.com/potplayer.html

MPC-HC

官网下载：https://mpc-hc.org/downloads/

对于懂行的朋友来说，MPC-HC绝对可以称得上是神级播放器。它的全称是“Media Player Classic Home Cinema”，最早是作为Windows Media Player(WMP)的同人作品开发出来的。然而，它却大大地超越了后者。

体积及资源占用很小，响应迅速，能播放网上几乎全部主流的音频视频格式，兼容性非常好，适合低配置老电脑上。

根据简单的比较，在同一台电脑上用 PotPlayer、QQ影音 以及 MPC-HC 播放同一部影片，基本上 MPC-HC 的资源占用都会节省不少，

而MPlayer WW 编译版则是由william0wang大佬编译的版本，相对原版，WW版Gui和配置界面都写的更好，核心也比原生态的优秀。

MPlayer WW 编译版对低配电脑支持比较好，MPlayer WW打开文件、拖动播放的速度快得不可思议，在播放高清时的CPU消耗也是较低的。

对于mx player，大家几乎有清一色的评论：安卓上最强播放器。

作为安卓平台上最强悍的视频播放器，MX Player以超强的解码性能以及兼容性闻名，对字幕的支持更是堪称一绝，能够兼容特效字幕，支持在线字幕匹配，看外语片无需自己找字幕！

Kodi

KODI中文网：http://www.kodiplayer.cn/

Kodi 支持解码播放几乎所有流行的音频和视频格式，3D、4K 高清什么的都没问题。

Kodi 专为大电视、大尺寸屏幕和投影优化的大字体全屏界面就不说了，但是 Kodi 之所以被称为神器的最主要的原因，还是在于它丰富强大的插件扩展！

自动下载电影字幕 / 更新电影封面和介绍信息等…..全球无数开发者为 Kodi 制作了大量实用的插件，如果你愿意折腾，Kodi 一定是你的首选。

Kodi 其实更适合在电视盒子上安装。一个软件，换你一个家庭影院。
