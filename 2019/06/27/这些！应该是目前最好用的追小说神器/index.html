这些！应该是目前最好用的追小说神器

发表于 2019-06-27 | 分类于 公众号 | 阅读数

阿虚之前写过一篇： 爱追书的你，可一定不要错过这四款软件！

因为有了不少更好的软件，所以这里重新写一篇，详细推荐一下

今天推荐的软件，主要是用来看网络小说。

IOS软件可以直接在App Store搜索软件名下载（阿虚也给出了扫码下载方法）

安卓软件下载方式请看文末，请看文末！！

一、阅读（安卓）

软件最起初由MonkeZ大佬独立开发，当时名为：MONKOVEL

Github开源地址：https://github.com/ZhangQinhao/MONKOVEL

后由Kunfei（坤飞）大佬改进在其基础上二次开发

Github开源地址：https://github.com/gedoor/MyBookshelf

  1. 软件免费开源，终身无广告
  2. 支持下载缓存

随着更新，这款软件不再直接提供书源了，需要用户自己导入

如果你不会用的话你可能会觉得：诶？这啥软件啊，咋啥都没有。

但其实这款软件强就强在能导入自制书源！

话不多说，阿虚提供400+的网络书源！！

二、爱阅书香（IOS）

这款软件，可以说就是IOS上的「阅读」了，功能基本上一样。

打开软件，和阅读一样，也是什么都没有。你需要自己导入书源。

在其软件的公众号上，提供一些常见问题解答和书源地址。

如果你不想去官方公众号上回复的话，阿虚帮你回复了，书源地址是这个：https://icc.one/2019/03/21/booksourcesImport/

用Safari浏览器打开上面的书源地址，就能直接点击导入书源了（下图左一）。

唯一的缺点就是这款软件的书源没有「阅读」那么多了

官方提供的所有书源导入全导入完的话，大概有上百个书源。

这款软件只有IOS版！安卓端建议用上面推荐的「阅读」！

建议趁早下载，避免哪天被下架了

三、快读全本小说（安卓 / IOS）

这款软件支持安卓、IOS

  1. 资源丰富，且全部免费看！（目前平台有150万本小说）
  2. 无广告弹窗、可换源！
  3. 支持横屏阅读
  4. 支持TXT导入阅读（IOS不支持）
  5. 支持下载缓存

这个软件看似没有换源功能，但是当你进入阅读界面的时候，其实是有换源功能的。

四、追书神器（安卓 / IOS）

应该很多人在用这款软件，所以阿虚还是分享一下（其实上面推荐的快读全本小说完全胜过追书神器了）

追书神器自从某个版本开始取消了“换源”功能，只能看追书神器提供正版书源，而正版书源需要开通VIP会员才能使用。

  • 安卓的话，阿虚提供一个去广告、去升级、可换源版本（由瀚海兰台大佬修改）

  • IOS的话，旧版追书神器依然是能换源的，IOS想装旧版APP请看阿虚之前写过的一期教程：5分钟教你安装IOS旧版APP

注意：安卓端请卸载已安装的版本，再安装此修改版本，否则可能出现安装失败等等情况发生。

五、藏书馆（安卓 / IOS）

软件官网：http://www.geeboo.com/

这款软件很聪明！允许用户上传电子书资源，别人可以借阅，变向规避了很多版权问题！所以上面的网络小说资源非常丰富！

而且不仅网络小说，很多出版读物都有资源。

一本书只能借阅15天，但是不担心，到期后可以续借！

小众作家的小说，在上面也能找到资源！

同时提供网盘功能，你可以上传电子书到网盘来云端观看！

这款软件支持安卓和IOS！
