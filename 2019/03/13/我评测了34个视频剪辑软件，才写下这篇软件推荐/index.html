一、手机端

尽管阿虚用的IOS，而且自己也不在手机上剪视频，但还是帮大家评测了大部分安卓软件。

手机上的视频剪辑软件可真的是太多了：AndroVid pro、Videoshop、FilmoraGo、VlogNow、大片、乐秀、Quik、接招、猫饼、一闪、HANI、ALIVE、inShot、小影、Cute CUT Pro、VidTrimPro、快影、巧影、快剪辑、Videoleap、DJI GO4……

阿虚也没能把市面上的软件都评测完，只评测了一些别人有在推荐的软件。

因为手机软件过多，就不以图片具体展示，请大家仔细看看阿虚的文字描述。

相信我，推荐的都是好软件。

推荐

以下推荐不分先后，因为各个软件都有一些特色功能，比如一键自动剪辑、去视频水印、支持美颜…又或者说可能这个软件的视频模板比较多，那个软件贴纸比较多。

具体使用的时候，可能得需要你1、2个软件结合使用。

HANI 哈你（安卓 / IOS）

官网：https://ihani.tv/index/page

  • 支持一键自动剪辑
  • IOS目前在设置里面可以免费内购去水印
  • 支持各种特效（可智能根据视频音乐节奏添加）
  • 支持调节视频颜色、速度、比例
  • 可添加视频文字标签、添加字幕
  • 可以视频倒放，加滤镜、字幕、音乐
  • 软件无广告，导出无水印

DJI GO4（安卓 / IOS）

其实不算一个视频编辑器，它只是大疆用来配合自家无人机和稳定器使用的配套软件。但是比起其他软件真的好用很多。

  • 支持一键自动剪辑
  • 支持（曲线）调节视频速度（1分钟视频以下可反转）锐度、色温、饱和度、亮度、对比度
  • 支持添加移轴、阴影、高光
  • 支持添加本地音乐（同时内置丰富的音乐）
  • 软件无广告，导出无水印

Quik（安卓 / IOS）

  • 支持一键自动剪辑
  • 支持添加文字
  • 支持视频原声混合背景音乐
  • 支持视频旋转、调色、加滤镜、剪切、分割、添加文字、变速（快中慢）
  • 软件无广告，导出无水印
  • 有2S左右的片尾（可利用手机自带的视频剪切功能把片尾剪掉）

以上三款软件都支持自动剪辑，这个自动剪辑功能可谓真·神器。

出去玩了一遭，拍了无数段视频，想最快的方式在朋友圈秀起来？

试试上面的三个软件吧

快剪辑（安卓 / IOS）

  • 支持视频滤镜、添加录音（剪辑时录音）、背景
  • 丰富视频模板
  • 支持添加马赛克（用于去视频水印等）
  • 支持添加模板文字、画中画、贴纸
  • 支持视频装饰（粒子类特效、魔法光效）
  • 视频变速、视频同框、多段拼接、音效
  • 支持无片头导出、无水印导出

Videoshop（安卓 / IOS）

  • 支持调整视频大小、背景、速度、调色
  • 支持添加录音（剪辑时候录音），视频修剪、倒放
  • 支持添加文本、贴纸、GIF贴纸、相册图片
  • 支持添加转场动画、滤镜、视频主题（新闻、老电影等，部分需付费）
  • 禁止软件的网络权限即可去广告
  • 导出时点击高级-去水印-会要求你给他们的Facebook点赞，点一下就行（网站打不开也不影响），切回软件就能导出无水印视频

VlogNow（安卓 / IOS）

  • 安卓只支持添加字幕、音频，加滤镜
  • IOS另外支持视频曲线变速、倒放、加背景、 变焦
  • 软件无广告、导出无水印

Inshot（安卓 / IOS）

  • 支持添加贴纸（提供大量贴纸）、文本、本地音乐（同时内置丰富的音乐）
  • 支持线性变速、视频旋转
  • 支持添加自定义视频背景
  • 支持视频水平反转
  • 软件无广告，导出无水印
  • 安卓有破解版，IOS去广告、视频水印需要收费，部分素材、贴纸、滤镜收费

简影（安卓 / IOS）

  • 全为视频模板，只需导入图片or视频即可一键制作
  • 部分模板收费，导出无水印版视频需要收费
  • 当然，以上收费功能，安卓有破解版，但IOS就无法享受了

乐秀视频剪辑（安卓 / IOS）

  • 视频模板多，视频主题多
  • 支持裁剪视频、压缩视频、视频转音频、格式转换
  • 画中画（收费）、加水印（收费）、马赛克（收费）
  • 支持视频转场、滤镜、音效、录音、贴图、视频特效
  • 支持添加多段音乐、字幕、滚动字幕（收费）、支持视频涂鸦（任意在视频上绘画）
  • 当然，以上收费功能，安卓版有破解版，但IOS就无法享受了

FilmoraGo（安卓）

  • 支持各种视频主题
  • 支持视频转场
  • 支持添加录音（剪辑时候录音）
  • 支持混音、贴图、画中画、视频旋转
  • 支持添加字幕、调速、特效
  • 软件无广告，导出无水印

AndroVid Pro（安卓）

  • 视频转GIF、转音频
  • 视频加滤镜、加文字
  • 修剪视频尺寸、提取部分视频、视频合并
  • 制作幻灯片
  • 提取视频某一帧
  • 软件无广告，导出无水印

VidTrimPro（安卓）

  • 支持视频转码
  • 支持视频修剪、合并、转音乐、抓取帧、加特效、旋转
  • 软件无广告，导出无水印

功能性的两款软件

PowerDirector（安卓）

即威力导演

  • 支持改变视频音量、视频翻转、倒放、片段复制、裁剪、变速
  • 支持添加音频、字幕、视频特效、贴纸（内置较少）、图片
  • 支持皮肤柔和器（即磨皮）
  • 软件无广告，导出无水印

VUE（IOS）

  • 登录后，设置里面可以免费关闭视频水印
  • 两种视频编辑模式：自由模式（不限时长和分段数）、分段模式（固定时长和分段数）
  • 支持视频美肤（中、强）
  • 支持滤镜、截取、分割、画面调节、变焦、倒放、镜头速度（0.25-4倍速）、原声增强
  • 支持贴纸、文字、字幕、音乐、录音

以上是支持美颜的两款软件

Videoleap（IOS）

  • 支持混合器（即视频混合图片）
  • 支持添加文本、音频、录音（剪辑时候录音）、滤镜（很多）、背景
  • 支持添加效果：镜头炫光、虚焦（收费）、棱镜、万花筒、填色（收费）、晕影、胶片颗粒
  • 支持毛刺（特效）：像素化、RGB（收费）、CRT、扫描、蚀刻、伪3D（收费）
  • 剪下框架、色调范围、（图层过多）调整IOS收费（1个月43,12个月253，一次性购买448）

不是特别推荐

不推荐的原因是无特色功能，或有一些功能限制。

iMovie （ Mac / IOS ）

官网：https://www.apple.com/cn/imovie/

  • 操作简单，只需导入视频片段，添加音乐、文字等
  • 提供了数十种滤镜特效及背景音乐
  • 支持加速、创建画中画及分屏视频

大片（安卓 / IOS）

  • 支持添加滤镜、文字、音乐（内置海量音乐）、录音
  • 支持视频变焦、裁剪、倒放、变速
  • 支持添加黑场、白场
  • 可在设置—通用里面免费关闭视频水印

拍大师（安卓 / IOS）

  • 安卓端强制要求安装「无名情报局」才能使用
  • 支持动态字幕
  • 视频变速、滤镜、添加音频、添加录音、视频转场（少）
  • 支持导出GIF动图（仅前10秒）
  • 不开会员，导出有水印

一闪（安卓 / IOS）

  • 登录后，可在设置里面免费关闭视频水印
  • 支持添加效果、文字、滤镜、音乐
  • 支持视频旋转、裁剪
  • 最长只支持导出280秒的视频

巧影（安卓 / IOS）

  • 专业度较高，操作难度与付费效果成为限制门槛，小白慎入
  • 商店大多素材、特效、模板、贴纸、动画均需VIP或单独付费，去水印需要VIP
  • 当然，以上收费功能，安卓版有破解版，但IOS就无法享受了
  • 变声器（Chipmunk、机器人、空灵、调制）
  • 音量曲线、颜色调整、暗角、视频模板、速度控制、旋转/镜像、滤镜画面调整、音视频淡入淡出

小影（安卓 / IOS）

  • 付费会员才能使用的功能：水印、1080P、无导出时长限制、无广告、视频参数调节、动画字幕、付费素材
  • 当然，以上收费功能，安卓版有破解版，但IOS就无法享受了

接招小视频剪辑（安卓）

  • 片段截取、调整尺寸、多段合成、添加水印、去除水印、旋转、添加音乐、变速、压缩、视频转GIF
  • 去除水印限制试用次数
  • 导出1080P超清画质需要VIP

乐剪辑（安卓）

  • 剪辑、拼接、配乐、倒放、比例、分割、变速、贴纸

二、电脑端

介于大多数人应该都是Windows系统，使用Mac以及Linux的人较少。

阿虚也是Windows系统，所以自己就不特别分享Mac和Linux软件了。

（但下面介绍的部分软件是跨平台软件，有需要的可以自己去其官网下载对应版本。）

以下就分入门、进阶、专业分开推荐

入门

快剪辑（WIN / IOS / Android）

360出品的快剪辑，官网：http://kuai.360.cn

▷阿虚自制的单文件版

无需安装，下载后双击直接使用。

▷ 打开软件会提示更新，请选择否

▷保存时可自行选择是否添加片头 or 水印（水印可自定义）

2.iDo（WIN）

优酷出品的iDo，最后更新停留在2016.01.20，不过软件依然可用。

▷阿虚自制的单文件版

无需安装，下载后双击直接使用。

▷打开软件后会提示脚本错误

这个不影响使用，随便你点击「是」或者「否」都行。

3. Apowersoft Video Editor Pro（WIN）

视频编辑王是Apowersoft开发的一款视频快速编辑软件，除裁切、合并、旋转、快进等功能外，其内置强大的特效，为视频添加注释、制作文字背景、字幕及编辑字体样式等，上手很容易。

▷阿虚自制单文件版

▷启动软件后请自行点击右上角设置语言

4.爱剪辑去片头片尾版（WIN）

  • 阿虚提供的是去广告、去片头、去片尾版

  • 界面排版给人感觉不是特别直观，但是花点时间能很快上手

  • 提供了非常多的 MTV 字幕、转场、文字、相框、贴图特效
  • 支持美颜，磨皮，加滤镜
  • 可制作卡拉 OK 文字跟唱特效

5.Shotcut（WIN / Mac / Linux）

  • 官网：https://www.shotcut.org/

  • 此软件免费开源的跨平台软件。

  • B站系列教程：https://www.bilibili.com/video/av10244743/

▷阿虚自制单文件版

阿虚提供单文件版，当然，你可以去官网下载绿色版或者安装版。

5.AVS Video ReMaker（WIN）

  • 官网：https://www.avs4you.com

  • AVS Video ReMaker是一款强大的视频剪辑软件，可以无需重新编码而切去视频的某些片段。

▷使用说明

使用前先运行 !)AVSVideoReMakerPortable.exe， 选择「安装」，会提示选择中文还是英文界面，然后执行主程序AVSVideoReMaker.exe。

6.iMovie（Mac / IOS）

  • 官网：https://www.apple.com/cn/imovie/

  • 苹果推出的视频剪辑软件，原来收费，现在免费，如果只是一般的剪辑，iMovie就足够了。

进阶

7.Vegas

  • 相比PR，Vegas就单独一个软件来说，功能比较全面

  • 由于界面设计原因，入门也比较简单。

8.EDIUS

  • 相比PR来说，PR功能更强大，Edius使用方便。
  • PR优点就是功能多，插件多，模版多，和Adobe家软件衔接的很好。
  • Edius的优点是界面友好，上手方便，对电脑要求低。
  • 阿虚提供Edius8.53傻瓜式一键安装版（内附安装教程视频）

专业

阿虚提供的版本都能免费安装使用，然后是否支持正版就看个人了。

9.Final Cut Pro（Mac）

Mac上的专业级视频剪辑软件，售价为299.99美元（国内售价为1998元）

10.Adobe Premiere（WIN）

这个就更不多解释了，虽然放在专业级，但软件使用其实挺简单的。

B站上搜索“PR教程”，你就能找到很多良心UP主做的系列视频教程。

▷阿虚提供两个版本的安装包

  • Adobe Premiere Pro CC一键安装版（160M）
  • Adobe Premiere Pro 2017（1.3G，内附安装教程）

PR2017和PR CC差别不大，CC具备PR所有基本功能，2017会稍微多一点点新功能。（阿虚我自己都用的CC。）

这个版本的PR CC是一件安装版，安装非常简单无脑。

比如你想把软件装在D盘，那就把压缩包解压到D盘，然后运行解压出来的“QuickInstall.exe”，就能一键安装完成了。

11.会声会影

会声会影由于版权问题，不多解释，下载地址到公众号上回复视频剪辑。
