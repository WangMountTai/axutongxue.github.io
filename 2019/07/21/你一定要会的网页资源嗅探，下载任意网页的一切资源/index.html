你一定要会的网页资源嗅探，下载任意网页的一切资源

发表于 2019-07-21 | 分类于 公众号 | | 阅读数

某网页上播放着一首歌，但是没提供任何下载按钮

那我们怎么下载？

阿虚之前有写过一篇：看完这篇教程，再没有下载不了的网页视频

今天这篇算是对于那一篇的补充，但其实今天的教程非常实用！能解决你对大多数网页的下载需求！

（另外就是，写了这篇教程，才方便阿虚写下一篇教程）

一、Chrome浏览器

首先想介绍的方法是Chrome浏览器 + Chrome浏览器扩展Chrono

（一）浏览器扩展Chrono

我们需要用到一款名叫Chrono的浏览器扩展

点击扩展图标，再点击下图所指的图标，来切换到Chrone的「资源嗅探器」功能

之后，在弹出的Chrono资源嗅探器中，则可以轻松的嗅探出当前网页上正在播放的音频

这时我们点击开始即可把文件下载到Chrono的默认下载目录

当然不止音频，大多数视频、图片都可以用资源嗅探器来获取轻松下载

备注

如果没有嗅探到下载地址：

  • 刷新页面后，确保你的音频 / 视频正在播放中，再点资源嗅探
  • 还是没有下载地址的话，可能是网站有防下载措施（多是视频类课程），这种视频想下载一般只能靠盗录

（二）怎么安装Chrono

方法1

如果你会富强上网，那当然是去谷歌的Chrome浏览器扩展中心安装了

你可以按下面的方法打开Chrome浏览器扩展中心

点击左上角的汉堡菜单栏，点击下方的「Chrome网上商店」

方法2

下载 Chrono.crx 扩展，离线安装

  • 关注微信公众号：阿虚同学

  • 回复：资源嗅探

  • 后台即会自动回复扩展下载地址

打开Chrome浏览器扩展中心

情况1

情况1是你下载阿虚提供的chrono.crx扩展，直接拖进Chrome浏览器的扩展中心就能安装了

情况2

但你可能会遇到安装时报错的问题，安装不了

这样的话，就用另外一个办法：把阿虚提供的Chrono.zip解压，然后点击Chrome浏览器扩展中心中的「加载已解压的扩展程序」，接着选择chrono这个文件夹就能成功安装了

方法3（新手推荐）

如果你看了上面的教程，还是觉得自己不会装这个chrono扩展，那阿虚只能再给了一个终极小白办法了：那就是直接下载自带Chrono的Chrome浏览器定制版

RunningCheese Chrome

RunningCheese Chrome是「奶酪」大佬提供的定制版Chrome浏览器

  • 绿色无需安装，下载解压即可使用

  • 可随着官方版本更新

  • 自带许多常用拓展以及拓展配置（IDM、油猴、Chrono…等等）

意思是你直接省去上面各种安装操作，你也不需要会科学上网什么的。

你直接下载这款定制版Chrome浏览器，所有东西都配置好了的，解压后直接使用即可

这款定制版Chrome浏览器下载安装地址，请直接到奔跑中的奶酪大佬的博客上去直接下载：https://www.runningcheese.com/chrome

二、傲游浏览器

傲游浏览器官网：http://www.maxthon.cn/

阿虚在用Chrome浏览器之前一直用的都是傲游浏览器，就是因为它有其他浏览没有的资源嗅探功能

当然，你可以去官网下载最新版。

  • 但如果不作为主力浏览器，只作为功能性浏览器不时使用一下的话，阿虚推荐一个自用的版本：傲游浏览器4.3.2.1000绿色版

  • 这个版本是阿虚找了很久才找到的，好不容易符合阿虚的要求：绿色版 + 资源嗅探功能正常使用

  • 反正阿虚现在是把这个版本的傲游浏览器当作一个下载器来使用了

使用方法

下面阿虚以傲游浏览器4.3.2.1000来演示下资源嗅探器的功能。

用傲游浏览器打开你想下载资源的网页，点击右上角的「资源嗅探器」

然后像下图一样，就能直接找到正在播放的音频的下载地址了

当然不止音频，大多数视频、图片都可以用资源嗅探器来获取轻松下载
