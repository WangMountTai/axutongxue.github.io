评测完国内外40个网盘，我来告诉你究竟哪个网盘好

发表于 2019-05-11 | 分类于 公众号 | | 阅读数

之前阿虚写过一篇《从前的阿虚也不备份文件，直到那天….》，提到了5T OneDrive注册方法。不过很多地址都失效了，所以重新写一篇关于网盘的推荐。

今天纯粹的阿虚自身经验分享，算是折腾网盘小半年来的自我总结吧。

没人看也无所谓，有人看了帮到你就算起到了我写这篇文章的目的吧。

一、总结

如果你不像阿虚一样喜欢分享资源，只是想找个好用的网盘的话，看看「一、总结」和「二、推荐」就应该足够了。

这篇文章的前提是阿虚折腾了国内外大大小小共约40多个网盘。

用一张脑图来总结就是下面这样：

二、先说推荐的

考虑到绝大多数人对网盘的需求不高。

大多数人的需求应该都是希望网盘免费+不限速+空间大，主要用于备份照片、电影、软件等。

所以阿虚做出如下推荐：

网盘 推荐用法 备注
OneDrive 个人版 同步盘，存储个人一般文件（软件等） 只是网页版被墙，淘宝2块钱可从5G扩容到15G
和彩云 同步盘 免费16G空间，官方很多免费获得空间活动，每月都有
坚果云 同步盘，存储办公文档（Word等） 免费用户上传流量1G/月，下载流量3G/月，空间受限于上传流量（会员￥30/月，￥199/ 年）
OneDrive for Business 存储影视 \ 音乐 \ 软件等 淘宝2-10块钱可以买一个5T帐号；或者购买Office365（￥398/年，可几个人合购）可获得1T空间

天翼云 备份电脑文件 电信提供，免费15G

解释一下同步盘

不仅仅是网盘那样存储东西的盘，顾名思义，同步盘的特点即是：本地与云端修改文件自动同步

  • 你修改了本地文件夹中的一个Word文件，“Ctrl+S”保存Word文件后的几秒，同步盘就自动上传到了云端！
  • 你删除了云端的文件，本地文件也会自动删除
  • 你删除了本地的文件，云端的文件也会自动删除（云端有回收站，所以本地误删的话可以去云端恢复）

阿虚用的就是One Drive同步盘

三、简介一下其他网盘

1.被墙

因为被墙的原因，就算你会科学上网，分享文件给别人也不方便。所以基本上只能是自己使用。

想要使用下面这几个网盘，科学上网是必须学会的。

Google Drive

官网：https://drive.google.com/drive/my-drive

支持：WIN / MAC / IOS / Android

  • 谷歌就不说了，被墙很久了。网盘和邮箱共用存储，免费15G。

MEGA

官网： https://mega.nz/

支持：WIN / MAC / IOS / Android / Liunx / Chrome 扩展

  • 只是网页版被墙（所以分享的文件也不能正常打开），但是各平台的客户端都可以正常使用！

  • 现在注册只有15G永久空间了，上下传均为1m/s左右

  • 下载文件时有流量限制的问题（这个需另外的软件来解决）

Pcloud

官网：https://pcloud.com/

支持：WIN / MAC / IOS / Android / Liunx / Chrome扩展 / Lightroom插件

  • 来自保加利亚的一个国外网盘，免费10G，淘宝可扩容到20G
  • 访问很不稳定，有时不翻墙又能访问，阿虚觉得基本上处在被墙边缘
  • 这个网盘有个好处是可以连接OneDrive并实现自动备份OneDrive文件（多一份保险）
  • 支持WebDAV

Yandex.Disk

官网：https://disk.yandex.com/

支持：WIN / MAC / IOS / Android / Liunx / Windows Phone

  • Yandex 是俄罗斯的最大搜索引擎服务提供商，地位就像是国内的百度，其提供的网盘可以说是很有保障了
  • 免费10G
  • 这个网盘很奇怪，网盘页面被墙了，但是文件分享页面却能正常打开，但下载速度一般
  • （以后阿虚可能会用这个网盘分享资源）

2 .小众没保障

这些网盘要么是个人运营的网盘，要么是公司不主营的网盘，很可能说没就没了，存点东西玩玩还可以。

ZZZ盘

网址：https://zzzpan.com/

这个地方其实是一个存种子的网盘，貌似是个人博主运营的网盘（他用来发布动漫资源）

小麦云盘

网址：http://own-cloud.cn/

500G空间，上下传不限速，最大5.0GB单文件上传，分享出去的图片，音乐，视频支持在线浏览

除了稳定性没保障，还真是个挺好的网盘

酷族网软

网址：https://www.kzwr.com/

无限空间，上传限制1G，只能单个文件分享，不能分享文件夹，分享出去的图片，音乐，视频支持在线浏览

网站运营了5年了，挺不错的一个网盘，但是由于是个人运营，所以还是没保障，随便存点东西玩玩还是行的

我的文档云

网址：http://docq.cn

应该是无限空间，没明确说明上传限制，阿虚测试大概是300M，只能单个文件分享，不能分享文件夹

背后是武汉快云网络技术有限公司，网站成立时间是2015年，稳不稳定大家自己斟酌斟酌吧

3.网赚网盘

怎么说呢，虽然说最稳定的网盘都是能赚钱的网盘，但是以下这些网盘其实都不推荐（主要是给别人的下载体验很不好）

实际上也就90网盘、城通网盘、128网盘能将就用用。

城通网盘

网址：https://www.ctfile.com/

怎么说呢，城通网盘虽然限速严重，但还是有一点好处的：因为分享出去的文件被人下载是有收入的，虽然依旧很多人在上传、分享资源；再如很多漫画、游戏、电子书资源只在城通上找的到（很多其他资源都失效了），所以才会有这样一句话：最稳定的网盘都是能赚钱的网盘。

但阿虚还是不会用这个网盘的，阿虚会选择其他网盘

免费用户注册即可获得400GB空间

免费用户都是上传到公有云，分享出去的链接90天没人下载的话文件会被删除，连续有人下载之后文件会被永久保存。

90网盘&128网盘

这两个网盘前端都很类似，但是90网盘做的要好一点

限制文件大小150M，相比蓝奏云，这两个网盘可以分享多级文件夹，这是最大的优点。

90网盘：http://www.90pan.com/

128网盘：http://www.128pan.com/

以下完全不推荐

  • 千脑云：http://www.qiannao.com/ （07年创始，但大概已经停止运行了，但还是能上传和分享文件；文件下载极慢）

  • 飞猫云：https://www.feemoo.com/（禁止任何音频、视频内容；满屏的误导充值VIPx选项，一般用户根本找不到普通下载地址）

  • UsersCloud：http://userscloud.com/（被墙了，国外的皮皮盘）

  • 皮皮盘：https://www.pipipan.com/（同城通 网盘旗下）

4.不能分享

中科院网盘

网址：http://ddl.escience.cn/pan/

中国科学院计算机网络信息中心提供的网盘，除了不能分享数据还是很有保障的

个人空间（同步版Beta）20G+个人空间10G+免费创建团队10G

和彩云

网址：https://caiyun.feixin.10086.cn

同步盘下载地址：http://caiyun.feixin.10086.cn/downloadClient.jsp

中国移动提供的网盘，默认是16G空间，和彩云APP上有丰富的活动，轻松免费获得百G空间。

拿来做电脑上的同步盘还是非常不错的！

和彩云不提供文件分享功能，但因为其和139邮箱绑定在一起，所以实际上可以用139邮箱来变相获得文件分享链接，具体操作可以看下阿虚之前写的那篇教程：从前的阿虚也不备份文件，直到那天….

DEGOO

网址：https://degoo.com/

德国网盘，服务很稳，起始空间100GB，用来备份手机图片非常不错！每天通过看10秒视频（+1G）和邀请好友（+3G），支持手机和电脑自动备份，上传和下载的速度都是全速状态，缺点是如果你84天没登陆，会丢失文件。

TeraCloud

注册网址：https://account.teracloud.jp/RegistForm.php/index/

如果你想注册，但又没用Chrome浏览器的翻译功能，可以看看这篇教程：https://moe.best/yoimono/teracloud.html

日本的一个网盘，访问速度还行，国内支持 WebDAV 协议的网盘只有坚果一家，速度很快且好用，可惜每月只给 1GB 的上传流量和 3GB 的下载流量，用起来很不爽。TeraCloud是其优秀的替代品。

免费空间10G，推广奖励政策： 5GB / 人，使用邀请码双方皆可额外得到 5GB 容量奖励。阿虚的邀请码：CREKA

如果您使用的是免费帐户，并且尚未登录官网90天，则TeraCLOUD上的帐户和数据将被删除。

以下不推荐

  • 比特球网盘：https://pan.bitqiu.com/（原来的盛天网盘，不看好这个网盘）
  • 京东网盘：http://jbox.jcloud.com/（京东还在测试中的网盘）
  • Zoolz：http://zoolz.com/（冷储存，只能拿来备份文件）
  • 亿方云：https://account.fangcloud.com/（实际上是一个企业网盘，算是大牌网盘了，免费50G空间，缺点就是无法分享）
  • UC网盘：UC浏览器内置，没有其他客户端，也没有网页版

5.分享型网盘

这类网盘都是提供一个二级域名供游客访问。

阿虚的储物间用的就是永硕E盘，是一个很有历史的网盘了，2005年注册的公司距今已经有14年了，很多政府都在用这个网盘，可以说很有保障了。

另外阿虚还找到了其他类似的网盘。

3ADISK

网址：http://www.3adisk.com/

是个个人开发网盘，2012年至今。

永硕E盘

网址：http://ys168.com/

免费用户提供1G空间

中国教育网盘

网址：http://www.edudisk.cn/

也是很有历史的一个网盘了，2006年成立至今。阿虚曾经预设的备份网盘，如果某天永硕E盘出问题了，阿虚会考虑搬迁到中国教育网盘

115网盘

网址：https://115.com/

资费详情：https://vip.115.com/

360网盘

网址：https://yunpan.360.cn/

资费详情：https://yunpan.360.cn/vip/product

7.无永久分享链接

分享出文件链接有限制，并不是永久分享链接。

###　沃家云

网址：http://www.wocloud.com.cn/

免费送500G，每日限制10G上传，下载速度1m左右，分享文件单次有效7天，不能续期

Naver Cloud

网址：https://nid.naver.com/（注册一个Naver邮箱账号）

网盘地址：https://cloud.naver.com

韩国门户网站（和百度类似）旗下的存储服务，网盘注册过程全韩语，你要是想注册可能得需要Chrome浏览器的翻译功能

空间30G，不付费的话分享文件单次有效期30天或者100次，不能续期（当然，得付费韩元）

163网盘

网址：https://mail.163.com/

一般能有7G空间，这是集成在163邮箱里的一个网盘，分享出去的文件只能下载20次，然后就会失效。

8.暂时没办法注册

STACK

这是荷兰的一个网盘，目前限制荷兰IP注册。阿虚也只注册到一个（电子书、字体阿虚就是用这个网盘分享的）

荷兰站注册地址：https://www.transip.nl/stack/

比利时站注册地址：https://www.transip.be/stack/

要是你想注册试试，可以看看这个2个教程

  1. https://www.zrj96.com/post-616.html
  2. https://www.cmsky.com/netdisk-stack/（仔细看下方评论）

9.其他还不错的网盘

OneDrive

个人版的网页版被墙了：https://skydrive.live.com/

OoneDrive分为个人版、教育版、商业版

教育版和商业版的登录入口：https://portal.office.com

  • 个人版是5G，淘宝2块钱可以升级到15G

  • 教育版是1T，注册需要edu邮箱，淘宝有卖（网上很多国外edu邮箱注册教程，但一般这种教程失效较快，所以阿虚在此也不给教程了；另外国外edu邮箱也是可能会翻车的，阿虚就翻车了不少国外edu邮箱）

  • 商业版是5T，OneDrive自助注册教程：https://51.ruyo.net/8263.html （注册一般是用临时邮箱来获取注册验证码，这个博主一直在更新可用的临时邮箱）

超星网盘

注册网址1：https://passport2.chaoxing.com/enroll?refer=http://i.mooc.chaoxing.com

注册网址2：http://passport2.chaoxing.com/register3?refer=http%3A%2F%2Fpan.ananas.chaoxing.com

注册需要填写学号，没有学号的话，填类似120014101010，应该就行了

超星公司是国内最早从事纸质资料的数字化以及制作电子出版物的公司之一，所以估计云盘储存也比较可靠。

在校学生应该都知道这个超星学习通，实际上网盘只是其附属功能，但是免费150G，而且上下传不限速可谓良心。

阿虚目前分享各种文件主要都是用的超星云盘（PC端无法获取分享链接，需要从手机APP获取）

还有个缺点就是不能分享文件夹。

天翼云

官网：https://cloud.189.cn/

现在注册只有免费15G了（阿虚是老用户，有2T）

分享的文件虽然需要审核，但是审核力度比百度云小。

文件下载是不限速的，文件上传限速，但是阿虚之前写过一篇教程来解决上传限速的问题：告诉你一条野路子，实现天翼云上传不限速

蓝奏云

官网：https://up.woozooo.com/

应该是挺出名的一个网盘了，上下传不限速，下载页面广告还能接受，不限空间

唯一的弊端就是，免费用户的文件限制上传大小100M；无法分享多级文件夹（只能一个文件夹一个一个文件夹分享）

微云

官网：https://www.weiyun.com

免费10G，从稳定性上来讲是绝对稳定的！开了会员可用离线下载功能。

阿虚也给过不限速下载的教程了：腾讯微云下载不限速技巧（这个方法还是有一定限制，短时间内下载过多会导致之后的文件下载失败）

因为和腾讯各种功能集成在一起，所以想实现不限速上传其实也很简单（比如传到QQ邮箱中转站，然后就能转存到微云；或者你QQ发送文件给自己小号，然后就能保存到微云了）

坚果云

网址：https://www.jianguoyun.com/

支持： Windows、macOS、Android、iOS、Linux 以及 Windows Phone 平台

免费版：上传流量1G/月，下载流量3G/月，空间受限于上传流量。支持增量备份！（就是说你一个word文件更新后，它会帮你保存一份历史版本）文件历史版本不会占用云盘容量，个人免费版能够保存一个月的文件历史版本；

资费详情：https://www.jianguoyun.com/s/pricing

AllSync

网址：https://cloud.allsync.com/apps/files/

支持： Windows、macOS、Android、iOS

荷兰网盘，支持WebDAV, 50GB免费空间，网盘是好网盘，唯一缺点是怕哪天可能墙。

AllSync的PC客户端是基于OwnCloud的, 可以在这里下载第三方客户端: https://github.com/owncloud/client/releases

四、阿虚在用的

其他还有很多杂七杂八的网盘啦，什么巴别鸟、燕麦云、曲奇网盘、蚁数，国外的BOX、dropbox、4Shared…阿虚就不再多加介绍了，感兴趣的可以自己谷歌一下。

最后说一下阿虚自己在用的配置

  1. Onedrive：WIN10上体验良好，15G 同步盘，用来同步基本上所有办公文件、常用绿色软件

  2. OneDrive 1T + OneDrive5T：主要用来分享各种影视资源，美剧、电影、动漫、音乐专辑、PPT模板、字体、贴图什么的…

  3. 超星云盘：150G，阿虚主要用来分享软件

  4. 永硕E盘 ：阿虚的储物间，这个就不多讲了

另外，阿虚没事喜欢在论坛，贴吧啥的帮人补链接（如果我有这个资源的话），这时一般用蓝奏云、酷族网软、城通网盘

目前部分音乐、漫画还在用90网盘分享，有粉丝吐槽广告多（因为阿虚忘了自己一直屏蔽了广告，失误），后期阿虚会迁移到其他网盘。

不过还是建议你看看：各平台浏览器去广告软件推荐
