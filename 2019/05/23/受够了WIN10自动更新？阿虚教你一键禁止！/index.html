受够了WIN10自动更新？阿虚教你一键禁止！

发表于 2019-05-23 | 分类于 公众号 | | 阅读数

遇到不少粉丝在问了，所以今天出个一键式禁止Win10更新教程

一、为什么要禁止Win10更新

  • 你正在WIN10上写材料，中途上了个厕所，回来竟然发现WIN10开始自动更新了！然而写的东西还没保存！！！

  • 曾经有段时间Win10升级补丁有BUG，导致许多人遇见：检测到更新→下载3、4个G的大补丁→开始升级→升级失败→撤销更新然后已下载的补丁自动删除→重启之后又会检测到更新…

凡事遇到过一次，我相信你一定会想要禁止Windows自动更新 的。

阿虚想说，自动更新本没有错，但是自动更新不可控制才是Win10最大的错误。

二、如何禁止Win10自动更新

Win10的升级途径很多，而且是是相互备份的。

也就是说无论是计划任务、易升的文件夹、微软升级服务、组策略，任意一点没有剔除干净的话，其他的几个也会跟着一起复活。

所以根据不同情况，你最好挨着检查一下阿虚下面说的东西。

1.卸载关于系统升级的软件

这里指的软件包括但不限于易升、Windows Update Assistant等，建议先看看自己电脑里有没有类似程序，如果有就删除。

避免你卸载的不干净，建议先看看阿虚之前写的一篇文章：阿虚是如何优化系统，清理垃圾文件的？

用阿虚之前分享过的专门的软件卸载工具来帮你彻底卸载软件。

2.删除升级相关文件

为了以防万一，打开C盘检查一下还有没有Windows更新相关文件没删干净。

先打开C盘

如果有Windows10Update这个文件夹，也删掉

接着打开C:\Windows

删除C:\Windows目录下面所有UpdateAssistant字样的文件夹

这种以UpdateAssistant开头的文件夹，是易升的助手

接着打开C:\WindowsS\SoftwareDistribution\Download

如果有下面类似于乱码的文件夹，大胆的全部删除即可（Download文件夹里的东西全是更新安装包）

2.使用Control Panel Plus

群里的小伙伴可能用过阿虚分享的群文件Anti Windows Update。

其实Control Panel Plus原软件名就是Anti Windows Update，作者最近新加入了诸多功能，并把软件改名为Control Panel Plus。

作者原话：方法更新于 2019 年 5 月 2 日，能有效阻止 1809 家庭版更新，永不失效。

软件使用非常简单

  1. 下载运行 Control Panel Plus（文末提供下载方式）
  2. 取消勾选Windows更新
  3. 右上角关闭软件
  4. 会提示是否重启使软件设置生效，重启一下即可

Control Panel Plus不止能用来关闭Windows自动更新，还能帮你一键关闭许多Windows不必要的功能。

当你把鼠标悬停在每个选项上时，软件会给出关闭or开启的建议。

  • 软件作者：知乎@知阳

  • https://zhuanlan.zhihu.com/p/33788671

三、更多办法

一般用了上面的办法，就基本上不会再出现自动更新的情况了。阿虚重装过很多次系统，按上面操作后都没再遇到过自动更新

当然，如果你依然担心Windows自动更新复活，我们还可以给它的棺材板再压两块石头。

1.把网络设为“按流量计费的连接”

右键桌面右下角的网络图标→打开网络和Internet设置

更改连接属性

选择你连接的网络，并把网络设置为按流量计费的连接

  • 需要注意的是，如果你换了网络（比如连了另外一个WiFi）你需要重新设置。
  • 如果你只用这个办法，Windows只会提示升级而不会自动升级。

2.禁止系统自动下载更新程序
