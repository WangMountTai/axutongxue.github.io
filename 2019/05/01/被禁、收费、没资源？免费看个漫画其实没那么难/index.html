被禁、收费、没资源？免费看个漫画其实没那么难

发表于 2019-05-01 | 分类于 公众号 | | 阅读数

今天重新写一下IOS / 安卓上的看漫画教程。

之前写过一篇：追漫画也是一个技术活，教你一个极少数人知道的看漫画方法！

但由于不久前动漫之家版权问题，导致了很多漫画看不了了（IOS端连APP都被下架了），所以重新更新一下，更新内容较多，建议重新看看

目录

  1. 网站
  2. 软件（安卓 / Win / IOS）
  3. 被禁漫画观看法
  4. 后记

一、网站

先推荐一些漫画网站，这些网站其实都可以像APP那样使用，具体可以看一下阿虚之前写过的教程：轻松把网页放在手机桌面，像APP一样使用

鼠绘漫画网

官网地址：https://www.ishuhui.com/

这个想必都知道的，鼠绘汉化组在翻译的漫画，网站上都有资源（一拳、海贼王、镖人、银魂、柯南等等）

土豪漫画

官网地址：https://m.tohomh123.com/

可看进击的巨人、尸兄等

漫画柜

官网地址：https://www.manhuagui.com/

可看哥布林杀手、海贼王、银魂、斗罗大陆、斗破苍穹等

漫画栈

官网地址：https://www.mkzhan.com/

自称国内原创漫画在线第一站，站内提供APP下载（Android / IOS）

漫画台

官网地址：http://www.manhuatai.com/

可看各大知名漫客的漫画：知音漫客（斗破苍穹）、风炫漫画（斗罗大陆）、神漫（绝世唐门）等等

极速漫画&动漫屋

日漫比较全（海贼王、一拳超人之类的都有）

极速漫画官网：http://www.1kkk.com/

动漫屋官网：http://www.dm5.com/

这类漫画网站一般广告都比较多，建议看看阿虚之前写的：全平台网页去广告软件推荐

二、软件（安卓 / IOS）

网易漫画、腾讯动漫（漫画）、有妖气、哔哩哔哩漫画…这些APP阿虚就不说了，有些漫画是其独家版权，很多独家漫画还是能免费看的。（但有了版权也可能就会出现收费才能看的问题）

坏坏猫（安卓）

软件下载方式见本文后记

这是一款集小说+漫画+搜索+阅读器的多功能老书虫漫迷的必备工具。

坏坏猫本身是不储存资源的，靠的是强大的云搜索功能，汇集全网多平台的海量资源。

遇到漫画无法正常打开或空白等情况，多半是漫画源不行了，点出菜单栏点击底部的「换源」即可切换不同的漫画源继续观看了。

异次元漫画（安卓）

软件下载方式见本文后记

这款软件本身不分享也不存储任何漫画资源，需要用户自己添加漫画书源才能进行搜索观看。打开软件默认是一个书源都没有的，因此什么功能都不能使用。

关于书源大家不会找不会制作也没关系，阿虚已经搜集了近百个可用的漫画源供大家使用，几乎可以说是包揽全网漫画资源了。

阿虚已将「软件」和「书源.txt」打包整理供大家下载了（下载方式见本文后记），打开软件——点击书源管理——点击右上角选择本地导入书源.txt

软件自身就非常干净没有任何广告，只要有能用的书源这款工具就能一直用到地老天荒。

多多猫（安卓 / WIN / IOS)

多多猫官网：http://ddcat.noear.org/

多多猫相当于是一个平台，可以安装漫画、轻小说、动画、周边、资讯、图集等二次元的站点的插件。得益于丰富的插件，多多猫的资源非常广，一个APP汇聚各大漫画网站资源。

  1. Android 版本 免费下载

  2. Win10 UWP版本：售价￥7

  3. IOS 版本（原「漫画.D」，已下架并不再维护，现只提供内测版软件，如果想安装，请在QQ搜索群（关键词：多多猫），安装方法请进群后自行看公告。安装需要手机越狱或不越狱用Cydia impactor 进行签名安装，具体安装方法请自行摸索，阿虚在这里不提供教程

Cimoc（安卓）

官网下载：https://blog.upx8.com/2304

这个软件和多多猫类似，内置11个漫画平台图源（有妖气、动漫之家、CC图库、汗汗漫画、动漫屋等），搜索时会在这11个平台上全部搜索一遍。

漫画人（安卓 / IOS）

官网下载1：http://www.1kkk.com/

官网下载2：http://www.dm5.com/

这个APP是由极速漫画（1kkk.com）和动漫屋（dm5.com）联合推出的一款APP，网站上有的资源，APP上都有，漫画资源还是挺全的。

其采用站外链接的方式来规避一些漫画的版权问题（比如海贼王，提供的就是百度贴吧观看链接）

可以看到，IOS失去动漫之家后，几乎没有免费看漫画的软件了（或许是阿虚不知道，如果你知道的话可以在文末留言告诉大家哦），漫画人可以看部分日漫，但许多漫画诸如东京喰种、进击的巨人上面还是看不了。

阿虚现在靠的是下面第三点要讲这个办法，来在IOS上看漫画。

三、被禁漫画观看法

这个方法稍微麻烦一点，但能看许多稀有的漫画，什么海贼、一拳超人、东京喰种、猎人、家教、斗罗大陆、伊藤潤二、暗杀教室等等都有。

方法就是：通过下载漫画后，推送到Kindle进行观看。

没有Kindle也没关系，Kindle APP 也是一样可以的！

APP支持WIN、MAC、IOS、Andriod，所以此方法就如上面所言：全平台通用

Kindle App + Kindle漫画网站

Kindle App下载：

MAC / IOS ：在App Store搜索「Kindle」即可下载

WIN：打开地址进行下载：https://www.amazon.cn/kindleapps

Android：在各大手机应用商店搜索「Kindle」即可下载

Kindle漫画站：VOL.MOE

网址：https://vol.moe/

银魂、东京喰种、进击的巨人、海贼王、浪客剑心、火影、死神…..

这个网站里不管是经典的，热门的漫画，还是小众的漫画资源都有，现共有5000+部漫画资源，网站由香港网友维护，有能力的小伙伴可以去捐助他们！

注册过程阿虚就不教：

注册好之后，我们点进任意漫画，就可以直接下载mobi格式的漫画了，或者直接推送到Kindle上。

当你订阅漫画后，有更新网站会自行推送到你的Kindle邮箱

如果你有Kindle：

  • 你可以在Kindle的内置浏览器打开网址：vol.moe，然后登陆账号下载漫画，下载完后漫画会自动加入Kindle书架

  • 看完上面那句话还不明白的，看下这个图文教程： https://www.zhihu.com/question/27741189/answer/146377533

如何推送漫画到Kindle

如果你之前没用过Kindle APP，不知道如何推送漫画到Kindle，你可以看一看我之前写的教程：如何获得手机最佳阅读体验？

更多kindle漫画网站

缤闹网

官网：http://www.binnao.com/

Ｋindle资源论坛，内有不少网友分享的优质漫画资源，免费注册&签到获得闹币用于回复获取下载地址

漫画补档：

官网地址：https://www.manhuabudang.com/

就和网站名字一样，众多完结漫画资源这个论坛上都有资源

Kindle漫画

官网地址： http://www.kindlecomic.net/

个人博主，分享一些小众漫画资源

Kindle漫画小站

官网地址：http://kindlemh.cc/

良心个人博主，许多小众漫画资源

四、 后记

IOS端失去动漫之家后，阿虚基本上完全转向用Kindle看漫画了。

所以以后阿虚也会继续分享一些自己下载的mobi格式漫画（目前储物间分享了部分漫画）

本期教程提到的所有网站，APP，我均会在之后更新到储物间之中，各位直接到储物间下载即可

当然，连载漫画更新最快的地方一般都是百度贴吧，追漫画的人应该都是知道的。

在版权意识逐渐加强的现在，很多漫画都被独家收购版权了，如果你有许多喜欢的漫画想收藏，或者说以后也想翻出来看，阿虚建议还是通过方法三——下载mobi格式漫画文件的方式来追漫画。

至少下载后，再也不用担心网站失效、找不到漫画观看地址等问题了。
