2019年5月，如何解决百度云限速问题？

发表于 2019-05-02 | 分类于 公众号 | | 阅读数

之前写过一篇：我评测了16款百度云破解版，就为告诉你哪款最好用！

但是那篇中提到的部分方法已经失效，且一些方法说明不到位，导致很多粉丝还是不怎么会操作，本文基于那一篇内容，更新一下。

这次，安卓 / IOS都有解决办法了！

以下提到的所有软件网站等，文末提供下载地址。

一、PanDownload（网页 / 安卓 / WIN）

PanDownload官网：http://pandownload.com

PanDownload 的作者真●良心又给力，近期分别发布了安卓版和网页版，可以说是最强的全平台百度网盘下载工具了。

1.网页版

网页版地址：https://www.baiduwp.com/

随着网页版的发布，PanDownload终于能说是一个全平台解决方案了！没错IOS也可以用！

有了网页版，不再需要软件，也就是说无论安卓、iOS、平板还是电脑，只要有浏览器能上网，就可以随时随地，不限速下载度盘文件了！

快捷使用方法

在文件分享链接「baidu」后加上 「wp」 ，按回车即可。

举例：

  • 原本分享链接：https://pan.baidu.com/s/axuhaoshuai

  • 修改后的链接：https://pan.baidu**wp**.com/s/axuhaoshuai

如果文件链接有密码，只需先填入密码，之后就可以正常下载。

需要注意的是：网页版限制了单个IP每天的下载次数，以防止滥用。

所以请不要使用IDM或迅雷等多线程下载器下载（会一次性耗尽）！开发者没有明确表示被限制的次数是多少。（大概是10 次）

网页版详情介绍：太意外！下载度盘文件不再需要客户端

2.安卓版

PanDownload 安卓版功能没 PC 版那么丰富，目前暂时还有一些限制，一句话总结就是：无需登录，高速下载，仅限 500M-2G 大小的文件（多了或者少了都不行，要么下载不了，要么加不了速）

安卓版详情介绍：手机上的不限速“百度网盘”来了

3.WIN版

由于百度云此前进行过更新，限制了不登录下载不了。

目前百度是针对账号进行限速，当一个非会员账号下载量达到一定阈值就会触发限速。账号被限速之后容易出现下载错误、掉连接数等问题，需要过几天或者开通会员才会恢复。

建议优先使用分享链接进行下载，可以降低被限速的风险。

被限速的账号可以尝试以下方法：

增加连接数

在PanDownload的软件设置中增加最大连接数

分享下载

分享需要下载的文件 -> 打开分享链接 -> 下载 （文件数量尽量控制在10个以内，太多的话容易出验证码）

打包下载

选中文件夹或多个文件 -> 提取链接 -> 复制下载链接到迅雷等下载器进行下载 （单个文件提取到的链接速度慢）

二、油猴脚本+多线程下载器（Win/ Mac）

此方法Windows和Mac都适用，且几乎不会失效(阿虚用了几年了)

建议下载2GB以内文件时使用，超过2GB最好使用其他方法。

缺点是提速有限，大约在500K/s到2M/s之间，有时候可以到5M/s以上，如果一个文件下载时间过长，比如超过一个小时，可能会下载失败需要重新下载。

1.安装暴力猴、百度直链脚本

安装浏览器扩展：暴力猴

不会这一步的同学看这篇教程：一个浏览器插件，和百度广告说再见

安装油猴脚本

安装好浏览器扩展之后，我们就可以添加各种浏览器脚本了。

打开GreasyFork（一个油猴脚本分享站）：https://greasyfork.org/zh-CN/scripts，或者按照下图操作也可以打开GreasyFork网站。

打开GreasyFork网页后，点击脚本列表，你将会看到下面这个页面，推荐安装这个百度网盘直链下载助手，作者是syhyz1990。

如果某天这个脚本失效，你可以自行尝试搜索安装其他百度网盘脚本（建议安装最近更新时间比较新的）

打开页面后，点击安装即可成功安装油猴脚本。

2.安装多线程下载器

多线程下载器其实很多

  • IDM（Win）
  • FDM（WIN）下载：https://www.freedownloadmanager.org/
  • Folx（MAC）下载：https://www.eltima.com/folx-download.html
  • Morix（WIN / IOS / Lunix）下载：https://motrix.app/

Windows推荐使用IDM，Mac上推荐使用Morix

下面阿虚讲一下IDM的安装使用注意事项等。

IDM安装说明

  1. 解压后，右键解压出来的文件「点我绿化卸载.bat」，选择管理员模式运行，安装软件。

  2. 打开Chrome浏览器的更多工具——扩展程序——打开右上角的开发者模式——把压缩包内的「IDMGCExt.crx」拖到浏览器进行安装

  3. 除Chorme谷歌、火狐浏览器之外，360、QQ、搜狗、UC、2345、百度、115、Cent、Yandex 等支持谷歌插件的浏览器安装方法类似第2步

  4. 不懂的话可以参考一下云盘万能钥匙的安装方法：http://ypsuperkey.meek.com.cn/helps/add-ypsuperkey-to-360browser

IDM建议进行如下设置

在软件左上角，点击「下载」-「选项」-「连接」

  1. 连接类型/速度设置为：「较高速率连接：局域网/WI-Fi/移动网络4G/其他」

  2. 最大连接数设置为：32

  3. 记得点击确定

3.这时再来下载

脚本安装好之后，我们再在网页上打开需要下载的百度云链接，即可看到下载助手按钮。

当你添加IDM的浏览器集成后，点击直接下载，即会自动调用IDM进行下载。

你也可以点击显示连接，使用其他下载器下载。

油猴脚本使用注意事项

  1. 由于百度云的措施更新，获取直链前需要在网页登录百度云帐号

  2. 点击「立即下载」前请等等等！等待IDM识别出文件名，识别出文件名后再点立即下载才行

  3. IDM识别不出来就刷新页面重新尝试

三、夸克浏览器（安卓）

阿里出品的夸克浏览器在安卓的应用商店都可以免费下载。

  1. 点击右下角—工具箱—UA设置，把浏览器标识改为PC；

  2. 返回首页点击右下角—下载—右下角，选择下载可调线程为最大64，点确定。

登录网页版百度云盘，选择电脑版页面，把文件分享出去后，在分享页面点击下载。

四、SpeedKoala BD云加速（WIN）

这款软件很久了，但却显得异常好用。2019年5月1号阿虚实测依然有效。

右键选择「超级加速下载（付费通道）」，阿虚测试下载1.3G视频，能跑满当前宽带（3M/S）

几点使用说明

  1. WIN10需使用管理员模式运行如果还不能运行，请使用WIN7兼容模式运行
  2. 电脑管家一类软件可能会提示“可以程序正在修改文档”（ariac2c.exe） ，选择允许即可。
  3. 需要安装环境 Microsoft .NET Framework 4.5.2，已有的略过；（文末提供下载地址）
  4. 需要安装环境 VC_RedistInstaller.exe，已有的略过；（文末提供下载地址）
  5. 如果下载速度不够理想，尝试在下载设置中设置：单任务连接数为32，单服务器最大连接数为16

五、Proxyee-down（WIN / MAC / Linux）

Proxyee Down 是一款开源的免费 HTTP 高速下载器，通过安装扩展实现特殊的下载需求（百度云限速）。

软件简洁，使用简单，具有多平台的优势，支持Windows、MAC、Linux。

  1. 在安装成功之后，进入网页版百度云，点击「下载」时会自动跳转到创建任务页面，然后选择保存的路径和分段数进行创建下载任务
  2. 下载、安装、使用、常见问题，作者都有详细在Github上说明，建议大家直接去官网查看阿虚就不再赘述了：https://github.com/proxyee-down-org/proxyee-down/blob/master/README.md

备注

作者表示暂停维护此项目（只是软件不继续更新，软件依然能正常使用），准备重新开发一款软件。

六、百度云单文件版等（阿虚特供）

阿虚之前提供的百度云本地VIP单文件版、百度云干净云、百度云听风暖都依然能正常使用。

（阿虚有次貌似被限速了，用了下面的客户端后就能高速下载了，貌似能解决帐号限速问题诶）

  1. 这三个软件均需使用QQ或新浪微博登录（若提示网络错误，请多次尝试登陆）
  2. 三款软件中更推荐百度云听风暖 and 百度云本地VIP单文件版

  3. 百度云听风暖为绿色版，下载后解压，直接运行即可（需进行部分设置，压缩包中阿虚都有说明）

  4. 百度云本地VIP版提速方式为缓慢提速，不用点击 免费试用加速通道，耐心等待即会提速至3M/S左右

绑定QQ或微博的具体操作步骤：

  • 打开百度网盘主页https://pan.baidu.com 然后退出。

  • 退出后，在登录界面重新点击下方QQ图标（或者微博）进行登录

  • 登陆QQ 成功后百度网盘会提示关联你的百度账号，这个时候再输入你的百度账号和密码即可成功绑定。

七、速盘（WIN）

之前，速盘由于被爆出在后台上传用户数据，一时间差评四起。

作者事后澄清是由于自身在代码上的疏忽造成的，并于之后进行了软件更新。

原贴链接：https://www.52pojie.cn/forum.php?mod=viewthread&tid=856862&page=6#pid23698112

速盘官网：https://www.speedpan.com/

速盘还是有一些特点的：比如不用登录百度网盘账号就可以高速下载百度网盘的分享链接，独有百度网盘资源搜索功能（搜索功能做的还不错）

目前，速盘分为免费版和极速版（收费），有不少坛友说收费了就变味了，关于这一点仁者见仁智者见智吧。

1.免费版

使用的是Aria2技术，完全免费使用，有几率触发黑科技加速，加速后同样可以满带宽下载网盘资源。

速盘免费版触发条件：

  1. 只能对分享链接加速；
  2. 只有当下载的文件超过1G才有几率触发加速。

免费版常见问题：

https://faq.speedpan.com/chapter3.html

2.极速版

使用的是速盘自己研发的下载引擎，按需付费使用加速器，速盘极速版没有触发限制，开始下载就有黑科技加速。

3.速盘公告

八、速盘+百度云官方版

使用速盘总会频繁出现「账号被限速」的问题。

现在给出一个和速盘官方FAQ不一样的 解决方案：

  1. 安装好最新版速盘（免费版）和百度云（官方版）
  2. 打开速盘，登录，找到需要下载的文件，把它分享出去，然后用这个分享链接去下载；果不其然，限速开始了，速盘开始频繁断流，提示“账号被限速”。
  3. 此时，把速盘的任务停止并取消，但是不要关闭速盘！启动百度云，正常去下载你要下载的文件；奇怪的事情发生了，百度云可以“速度不慢”地下载文件了（也许1~2MB/s，也许5MB/s，不能跑到你梦想中的几十MB的速度，但是总比10kb/s好多了吧）

备注

  • 这种现象不是100%会出现，但是大概率会出现，大概是每下7~8个文件（一集1G左右视频文件）会“正常地龟速”一次

  • 百度云的VIP加速“免费试用”选项，可点，也可不点，此操作与上述现象没有必然联系

方法来自吾爱破解论坛：Caitingting
